Inequality Comparison Operator (!=) let a = '5'; let b = 5; console.log(a != b); // Output: false c) comparision operator examples : > , < , >= , <= , === 1.Greater Than (>) let a = 10; let b = 5; console.log(a > b); // true 2.Less Than (<) let a = 3; let b = 5; console.log(a < b); // true 3.Greater Than or Equal To (>=) let a = 7; let b = 7; console.log(a >= b); // true 4.Less Than or Equal To (<=) let a = 3; let b = 5; console.log(a <= b); // true 5.Strict Equality (===) let a = 5; let b = '5'; console.log(a === b); // false, because the types (number and string) are different d) Logical operator examples : && , || 1. AND (&&) Operator The && operator returns true if both operands are true, otherwise, it returns false. Example: let a = true; let b = false; console.log(a && b); // Output: false (because b is false) If both a and b are true: let a = true; let b = true; console.log(a && b); // Output: true 2.OR (||) Operator The || operator returns true if at least one operand is true, and false only if both operands are false. Example: let a = true; let b = false; console.log(a || b); // Output: true (because a is true) If both are false: let a = false; let b = false; console.log(a || b); // Output: false 3. NOT (!) Operator The ! operator is used to invert a boolean value. It returns true if the operand is false, and false if the operand is true. Example: let a = true; console.log(!a); // Output: false (because a is true, !a is false) If a is false: let a = false; console.log(!a); // Output: true (because a is false, !a is true) e) sorthand operator examples : +=, -=, *=, %=, /= etc f) increment/decrement operator examples : ++ , -- etc g) ternary operator examples : ?:
